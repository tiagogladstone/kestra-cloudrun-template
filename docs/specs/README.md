# Especificações de Features

Esta pasta contém os documentos de especificação gerados pelo workflow `/planejar-feature`.

## Como usar

1. Execute `/planejar-feature` com o Agente IA
2. O Agente vai criar um arquivo `[nome-da-feature].md` aqui
3. Revise e aprove antes de prosseguir para `/criar-banco`

## Estrutura de cada spec

- Resumo da feature
- Modelagem de dados (SQL Draft)
- Diagrama de fluxo (Padrão A, B ou C)
- Lista de tarefas (Checklist)
